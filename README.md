# Company_Analysis
This project is about analyzing different companies in India, mainly in the IT sector. It shows the difference between private and public companies and compares their number of offices.It explains that IT services and consulting companies have the highest number of employees.
This project is about analyzing companies in India, mainly focusing on the IT sector. The main goal of this project is to understand how companies are performing, how they are distributed, and how the IT industry is growing over time.

First, the project compares private and public companies. It shows that public companies have more offices (739) compared to private companies (501). This means public companies are more spread across different locations.

Next, the project talks about top companies based on reviews. Companies like Accenture and Cognizant have the highest number of reviews, which shows they are well-known and have many employees sharing feedback. Wipro and Infosys also follow them.

The project also explains most demanding companies. It shows that IT services and consulting companies are the most popular and have the highest number of employees (around 10 lakh), while other companies have fewer employees (around 2–3 lakh).

Another important part is state-wise analysis. It shows that Maharashtra has the highest number of IT companies (31.58%), followed by Karnataka and Uttar Pradesh. This helps us understand where IT companies are mostly located in India.

The project also studies the growth of the IT sector from 1999 to 2017. It shows that growth was highest in 2000 (41%), then reduced to around 11% for some years, and later increased again to 17% in 2017. This shows the industry is growing steadily over time.

For analysis, the project uses Excel tools like filtering, grouping, charts, percentage calculation, and trend lines. These tools help to present the data in a clear and simple way.

In conclusion, this project helps us understand company distribution, top companies, and IT industry growth in India. It also suggests future improvements like adding salary data, using Power BI for dashboards, and collecting real employee feedback for better analysis.
